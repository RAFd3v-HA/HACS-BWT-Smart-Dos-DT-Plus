

# HACS-BWT-Smart-Dos-DT-Plus

Lokale HACS Integration für BWT Smart Dos DT Plus.

## Features

- Lokale API
- Config Flow
- Sensoren
- Fehlerstatus
- Restvolumen
- Dosierrate
- Firmware Informationen
- Wasserverbrauch in Litern

## Installation

### HACS

1. Benutzerdefiniertes Repository hinzufügen
2. Repository Typ: Integration
3. Home Assistant neu starten

## API Aktivierung

Die API muss in der BWT App aktiviert werden.
